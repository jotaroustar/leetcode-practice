## 已掌握解题思路

- 空间换时间（哈希表）
- 双指针·快慢型（原地修改）

- 字符串处理（切片/匹配）
- 位运算（异或）
- 二分替换（部落阵列O(log n)）
- 滑动窗口（最短/终止子队列）
- 快慢指针（环检测/环入口 a=c）
- 递归（树的深度/翻转，后序/前序遍历）
