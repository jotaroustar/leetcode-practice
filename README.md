## 已掌握解题思路

- 空间换时间（哈希表）
- 双指针·快慢型（原地修改）
- 双指针·对撞型（背包/出售）
- 字符串处理（切片/匹配）
- 位运算（异或）
- 二分替换（部落阵列 O(log n)）
- 滑动窗口（最短/终止子队列）
- 快慢指针（环检测/环入口 a=c）
- 递归（树的深度/翻转/对称/平衡/直径，DFS后序/前序）
- 多层递归（子树判断，复用 isSameTree）
- BFS（层序遍历，deque + len(queue) 固定层数）
- 栈（后进先出，基于三元表达式/占位符的防御性安全弹栈）
- 双栈单向倾倒（利用惰性双栈实现 FIFO 队列）
